### 逐题加工

#### 📌 课15.1 → [二阶非齐·待定系数]

**为啥没思路**:右端是 `x² + 8e^(2x)` — 两个不同型自由项,要**分别**求两个特解再相加。

**入手信号(铁律)**:
> 右端见 `P(x) + e^αx·Q(x)` → 拆成两个特解 y* = y₁* + y₂*

**口诀**:"**多型分治**" — 几种型设几个特解,最后相加

**关键**:特征根 r=2(二重) → e^(2x) 这一项要乘 **x²**(重数k=2),所以选 (B) `ax²+bx+c+dx²e^(2x)`。

📝 框架图标:`[二阶非齐·待定]` 旁 → "课15.1·多型分治⚠️重数定x^k" 🔵

#### 📌 课15.2 → 🆕[二阶非齐·解的结构]

**为啥没思路**:这题压根不要解方程!只用**解的结构定理**。本节点框架里漏了。

**核心套路**:
> 给N个非齐次特解 → **两两作差**得到齐次解 → 凑通解

具体:
- y₁-y₃ = e^(3x) (齐次解1)
- y₂-y₃ = e^x   (齐次解2,与上面线性无关)
- 通解 = C₁·e^(3x) + C₂·e^x + (-xe^(2x))

代初值 y(0)=0, y'(0)=1 → C₁=1, C₂=-1
特解:**y = e^(3x) - e^x - xe^(2x)**

**入手信号**:"给2个或3个非齐次解" → 立刻作差找齐次解  
**口诀**:"**差是齐次,通解=齐通+任一特**"

📝 框架图标:`[二阶非齐]` 下新建子节点 → "解的结构·课15.2·作差找齐通" 🔵(新节点)

#### 📌 课15.5 → 🆕[一阶线性·整体代换]

**为啥换不出来**:你想"令u=啥",但这题不是替换变量,是**把 e^y 当成新未知函数 u**。

**核心套路**:
> 见方程含 `e^y` 或 `y^n`(且不是伯努利标准型) → 令 **u=e^y** 或 **u=y^n** 化为关于u的一阶线性

具体推导:
- y' + 1 = e^(-y)·sin x
- 两边乘 e^y:  e^y·y' + e^y = sin x
- 注意 **(e^y)' = e^y·y'**(对x求导链式)
- 令 u=e^y → **u' + u = sin x** ← 一阶线性!直接套公式
- 解得 e^y = e^(-x)[½(sin x - cos x) + C]

**入手信号**:"方程含 e^y / e^(-y)" → 令 u=e^y  
**口诀**:"**整体当u,链式求导**"

📝 框架图标:`[一阶线性]` 下新建子节点 → "整体代换·课15.5·u=e^y" 🔵(新节点)

#### 📌 课15.10 → 🆕[一阶线性·变限积分求导]

**为啥换不出来**:这题根本不是"换元",是**对方程整体求导**消掉积分号。卡在"看到二重积分不知道咋办"。

**核心套路三步**:
> 1. 二重积分 → **极坐标**化为变限定积分
> 2. **对方程求导**消掉积分号(变上限求导公式)
> 3. 得到一阶线性ODE,套公式

具体:
- ∬f(½√(x²+y²))dxdy = 2π∫₀^(2t) r·f(½r)dr (极坐标)
- f(t) = e^(4πt²) + 2π∫₀^(2t) r·f(½r)dr
- 两边对t求导(上限是2t,要乘2): f'(t) = 8πt·e^(4πt²) + 8πt·f(t)
- 一阶线性: **f'(t) - 8πt·f(t) = 8πt·e^(4πt²)**
- 初值 f(0)=1(代t=0,二重积分区域为0)
- 解得 **f(t) = (4πt²+1)e^(4πt²)**

**入手信号**:"f(x) = ... + ∫含f的积分" → 求导消积分号  
**口诀**:"**含∫的方程,求导降ODE**"  
**⚠️易错**:上限是 φ(t),求导要乘 φ'(t)

📝 框架图标:`[一阶线性]` 下新建子节点 → "变限积分·课15.10·求导消∫" 🔵(新节点)
        旁加红笔 → "⚠️上限φ(t)求导×φ'(t)" 🔴

#### 📌 课15.13 → [几何应用]+[缺x降阶] (综合大题)

**为啥算错**:算法链长(列方程→求导→降阶→分离),任一步错全盘错。

**完整链路 + 关键节点**:

1. **几何建ODE**: 
   - 子切距 |TP| = y/y',故 △MTP面积 = ½·(y/y')·y = **y²/(2y')**
   - 曲线下面积 = ∫₀^x f(t)dt
   - 比例3:2 → 4y'·∫₀^x f(t)dt = 3y²

2. **求导消积分**: 
   - 两边对x求导: 4y''·∫f dt + 4y'·y = 6y·y'
   - 代回 ∫f dt = 3y²/(4y'),化简得 **3y·y'' = 2(y')²** ⚠️这步最容易错

3. **缺x型降阶**: 令 y'=p, y''=p·dp/dy
   - 3y·p·(dp/dy) = 2p² → 3y·dp/dy = 2p (约掉p,因 y'>0)
   - dp/p = (2/3)dy/y → p = C·y^(2/3)

4. **再分离**: y^(-2/3)dy = C·dx → 3y^(1/3) = Cx → **y = kx³**(由 y(0)=0 定常数)

**入手信号**:"几何题+含面积/弧长积分+面积比" → 求导降阶  
**口诀**:"**列→导→降→分**" 四步走

#### 📌 1000-10 → [一阶线性·定积分通解] · 完整套路 ★

dy/dx + y = φ(x), y(0)=0, |φ|≤k, 证 |y(x)|≤k(1-e^(-x))

**完整三步(任逸两步都没想到)**:

**Step 1 · 凑全微分(=积分因子法)**
- 两边乘 e^x:e^x·y' + e^x·y = e^x·φ(x)
- 左边正好是 **(e^x·y)'** (积/导反算)

**Step 2 · 不定积分→变限积分**
- 不定积分写法:e^x·y = ∫e^x·φdx + C ← 普通通解
- **变限积分写法**(用初值):**两边从 0 到 x 积分**
  - e^x·y(x) - e^0·y(0) = ∫₀^x e^t·φ(t)dt
  - y(0)=0 → **y(x) = e^(-x)·∫₀^x e^t·φ(t)dt**

**Step 3 · 估计**
- |y| ≤ e^(-x)·∫₀^x e^t·k dt = e^(-x)·k·(e^x-1) = **k(1-e^(-x))** ✓

**入手信号铁律(本讲第二大盲区)**:
> 求 **|y|≤... 有界证明** / **求极限** → 通解必须写成**定积分(变限积分)**形式

给三个特解 y₁=xe^x+e^(2x), y₂=xe^x+e^(-x), y₃=xe^x+e^(2x)-e^(-x),求**方程**

**待定系数法为啥算吐:**

- e^x 部分:λ=1 二重根,要乘 x²,设 y₂*=Dx²e^x,二阶导含 (2+4x+x²)e^x,代回算

**微分算子法 — 5 行解决:**

**对 6x²(λ=0,F(0)≠0):**

**对 -4e^x(λ=1,F(1)=0,二重根):**
- **二重根公式: 1/(D-α)² · e^(αx) = (x²/2!)·e^(αx)**
- → 1/(D-1)²·(-4e^x) = **-2x²e^x**

**通解**: y = (C₁+C₂x)e^x + 6x²+24x+36 - 2x²e^x

**算子法核心公式表(贴框架图)**:

| 自由项 | 公式 |
|--------|------|
| e^(αx),F(α)≠0 | 1/F(α)·e^(αx) |
| e^(αx),F(α)=0 单重 | x/F'(α)·e^(αx) |
| **e^(αx),F(α)=0 k重** | **x^k/k!/F^(k)(α) · e^(αx)** |
| 多项式 P(x) | 1/F(D) 展成幂级数,截到 deg P |
| e^(αx)·P(x) | e^(αx)·1/F(D+α)·P(x) (位移定理) |
