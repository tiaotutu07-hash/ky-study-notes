## 4. 概念对照(精简,只列有差异)

| 概念 | 王道 | 竟成 | CSAPP |
|---|---|---|---|
| Cache 单位 | 行/块等价混用 | 行/块等价(有提示) | **严格:块=信息包,行=容器(含辅助位),组=多行集合** |
| 写命中策略1 | 全写法(直写,Write Through) | 全写法/直写法/write through | 直写(write-through) |
| 写命中策略2 | 回写法(Write Back) | 写回法(write back) | 写回(write-back) |
| 写不命中策略1 | 写分配法(Write Allocate) | 写分配法 | 写分配 |
| 缺失分类 | 只提抖动 | 只提抖动 | **三分类:冷/冲突/容量** |
| Cache 容量 | 总容量 = 数据 + 辅助位 | 同王道 | C = S × E × B(数据区,不含辅助位) |
| 命中率 | H | p | hit rate |
| 平均访问时间 | Ta = Tc + (1-H)·Tm | Ta = Tc + (1-p)·Tm | (公式等价) |
| 组相联路数 | r 路 | k 路 | E 路 |
| 组数 | Q | (未明确符号) | S |
| 总位数符号 | c=log₂行数, b=log₂块大小, q=log₂组数 | M, N | s=log₂(S), b=log₂(B), t=m-s-b |

**study.md 用语决策:**
- 主用王道叫法,首次注英文
- **统一参数符号**:全文用 **m / s / b / t / S = 2^s / E / B = 2^b**(CSAPP 风格,但在直接映射时桥接说明"王道 c = s")
- 行/块/组在讲组相联前用 CSAPP 严格定义(铁律6),其后日常允许"行=块"等价
